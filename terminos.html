Términos y Condiciones

Última Actualización: 2 de abril de 2025

1. Aceptación de los Términos

Al acceder o utilizar los servicios de R2D1 BOT, el usuario acepta cumplir con estos términos y condiciones. Si no está de acuerdo con alguno de los términos, debe abstenerse de utilizar nuestros servicios.

2. Servicios Proporcionados

R2D1 BOT ofrece servicios de agendamiento de citas y gestión de calendarios mediante integración con Google Calendar. El usuario puede utilizar nuestros servicios para gestionar sus citas de forma eficiente. Además, el servicio incluye el uso de la API de Meta (Facebook), que permite la integración con plataformas de terceros, en conformidad con los términos y condiciones de Meta.

3. Uso del Servicio

El usuario se compromete a utilizar los servicios de manera responsable y conforme a la legislación vigente, incluyendo las normativas europeas de protección de datos (RGPD) y las condiciones de uso de las plataformas asociadas (como Meta en el caso de integrar el servicio con Facebook).

4. Información Recopilada

Recopilamos la siguiente información personal:

  • Datos proporcionados por el usuario (nombre, número de teléfono, dirección de correo electrónico).
  • Datos de uso relacionados con la interacción del usuario con nuestros servicios.
  • Datos técnicos (dirección IP, dispositivo, sistema operativo).
  • Datos de la API de Meta (Facebook)
